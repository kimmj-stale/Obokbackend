# 📚 Obok 📚
## 
❓ OUR VISION
**중고등학생이 하루하루의 공부를 성찰할 수 있도록 학습일기를 기록하는 앱을 제공하기**

공부에 흥미나 열정을 이제 가져보려는 모든 중고등학생이 스스로 학습의 의미를 발견하고, 지속 가능한 성장을 이루는 주체적인 학습자가 되는 세상을 만든다.
